# Club Fantasy MX (Demo)

Sitio web estático (HTML/CSS/JS + Bootstrap) para el **Proyecto Integral de Ingeniería de Software**.
Incluye: Inicio, Catálogo, Carrito, Cotización, Pedido, Modificar/Eliminar (demo) y Panel Administrador (demo).

## Cómo publicar en GitHub Pages
1. Crea un repositorio en GitHub, por ejemplo: `club-fantasy-mx`
2. Sube **todo** el contenido de esta carpeta.
3. En GitHub: **Settings → Pages**
4. Source: **Deploy from a branch**
5. Branch: `main` y folder: `/root`
6. Tu link quedará como: `https://TU_USUARIO.github.io/club-fantasy-mx/`

> Nota: El carrito y pedidos se guardan en `localStorage` del navegador (sin backend).


## Nota
En el menú **Pedido** se incluyen accesos a **Modificar** y **Eliminar** como pide el documento.


## Imágenes
Las imágenes del catálogo están en `img/playeras/` y se enlazan desde `js/app.js`.
